The Code for Racecar is structured in 5 functions. 


                      function getValue() {
                        // Get the string from the user.
                        let userString = document.getElementById('inputString').value;
                    
                        let userStringNoSpecial = removeSpecials(userString);
                    
                        let reversedString = reverseString(userStringNoSpecial);
                    
                        let palindromeChecked = checkPalindrome(
                          userStringNoSpecial,
                          reversedString
                        );
                    
                        displayPalindrome(palindromeChecked);
                      }
getValue()

The function getValue() stores the string from the user input in a variable called userString.

This function then calls the remaining functions in order of execution, to remove special characters, reverse the string, check if it's a palindrome or not, and finally display the string. 


                      function removeSpecials(userString) {
                        // Remove Special characters and make the string lowercase.
                        const noSpecials = userString.replace(/[^a-zA-Z]/g, '');
                        const lowercase = noSpecials.toLowerCase();
                    
                        return lowercase;
                      }
removeSpecials()

The removeSpecials() function has the responsibility of removing special characters and spaces from the user input.

In order to remove these characters, we use a regex pattern to whitelist a certain set of characters and use the replace() function on our userString and store the result in a new variable called noSpecials.

Now we call the toLowerCase() function on the new noSpecials string and store it in a new variable called lowercase.

Finally we return lowercase.


                      function reverseString(userStringNoSpecial) {
                        let reversedString = '';
                        // Reverse the string.
                        for (let i = userStringNoSpecial.length - 1; i >= 0; i--) {
                          reversedString += userStringNoSpecial[i];
                        }
                        return reversedString;
                      }
reverseString()

The function reverseString() loops over the length of the new userStringNoSpecial, which has been regexed and lowercased in the previous function, and loops in reverse for the length of the string, and adds each character to a new string called reversedString.

Finally we return the reversedString.


                      function checkPalindrome(userStringNoSpecial, reversedString) {
                        // Check string for palindromation (yes that's now a word).
                        const palindromeMessage = `The word ${userStringNoSpecial} is a palindrome and becomes ${reversedString}!`;
                        const errorMessage = `The word ${userStringNoSpecial} is not a palindrome, please try again with a different word or string.`;
                    
                        if (userStringNoSpecial == reversedString) {
                          return palindromeMessage;
                        } else {
                          return errorMessage;
                        }
                      }
checkPalindrome()

In the function checkPalindrome() we take in the userStringNoSpecial and the reversedString as parameters.

We declare two new messages and store them in variables for use in our if check.

Finally we check if the strings (parameters) match and if they do, we return the palindromeMessage, and if it doesn't (else) we return an errorMessage, asking the user to try again. 


                      function displayPalindrome(palindromeChecked) {
                        // Display the returned string.
                        document.getElementById('results').innerText = palindromeChecked;
                      }
displayPalindrome()

The function displayPalindrome() takes in the variable containing the message from our checkPalindrome() function, as a parameter.

We use that parameter when inserting the text from this variable into the text of the "results" id on the page.
